# Финальный отчет по ML-проекту

## 1. Название проекта
Предсказание цены краткосрочной аренды по данным объявлений Airbnb.

## 2. Бизнес-задача
Построить модель регрессии для прогнозирования цены объекта аренды на основе характеристик жилья, хоста, локации, доступности и отзывов. Такая модель может быть полезна для оценки конкурентной цены, поддержки хостов и анализа рынка.

## 3. Общая информация о данных
- Размер датасета: 15000 строк и 36 столбцов.
- Предметная область: объявления Airbnb-style для краткосрочной аренды.
- Каждая строка соответствует одному объекту аренды и связанным с ним атрибутам хоста, локации, доступности, отзывов и цене.
- Общее число пропусков до подготовки: 31176.
- Дубликаты: 0.

## 4. Целевая колонка и тип ML-задачи
- Целевая колонка: `price`
- Тип задачи: regression

## 8. Созданные признаки
- beds_per_accommodates — число кроватей на гостя.
- accommodates_per_bedroom — отношение вместимости к числу спален.
- reviews_per_month_per_review — связь текущей активности отзывов и накопленного числа отзывов.
- availability_ratio_30_365 — сравнение краткосрочной и годовой доступности.
- host_listing_concentration — доля активных объявлений среди всех листингов хоста.
- review_scores_mean — средняя оценка по ключевым рейтингам.
- description_char_len — длина описания в символах.
- description_word_count — количество слов в описании.
- amenities_item_count — число перечисленных удобств.
- property_type_char_len — длина текстового типа жилья.
- host_tenure_days — давность хоста.
- days_since_first_review — давность первого отзыва.
- days_since_last_review — давность последнего отзыва.

## 14. Бизнес-интерпретация результатов
Модель объясняет около 60% вариации цены, что указывает на полезный, но не идеальный уровень качества для прикладного сценария оценки стоимости. Ошибка в RMSE около 127.36 означает, что для части объектов модель может заметно отклоняться от фактической цены, поэтому результат стоит использовать как ориентир, а не как единственный источник решения.

Для бизнеса это означает следующее:
- модель может помогать в первичной оценке рыночной цены;
- полезна для мониторинга аномально дорогих или дешевых объявлений;
- может использоваться как вспомогательный инструмент для хостов и аналитиков;
- для задач, где критична точность ценообразования, потребуется дальнейшее улучшение признаков и/или моделей.
