# Финальный отчет по ML-проекту: прогнозирование цены Airbnb

## 1. Бизнес-задача
Задача проекта — построить модель регрессии для прогнозирования цены объявления Airbnb по характеристикам объекта, хоста, отзывам и признакам доступности. Такая модель может использоваться для оценки рыночной стоимости объекта, поддержки ценообразования и анализа факторов, влияющих на цену.

## 2. Общая информация о данных
- Объем данных: 15000 строк, 36 исходных признаков.
- Предметная область: Датасет, вероятно, относится к объявлениям Airbnb и характеристикам объектов краткосрочной аренды в разных городах.
- Интерпретация строки: Одна строка, вероятно, описывает одно объявление/объект аренды с его параметрами, расположением, отзывами и ценой.
- Пропуски в исходных данных: 31176.
- Дубликаты: 0.

## 3. Целевая колонка и тип ML-задачи
- Целевая колонка: `price`
- Тип задачи: `regression`

## 4. Группы признаков из схемы
- numeric: accommodates, bathrooms, bedrooms, beds, host_listings_count, host_total_listings_count, latitude, longitude, availability_30, availability_60, availability_90, availability_365, number_of_reviews, number_of_reviews_ltm, number_of_reviews_l30d, review_scores_rating, review_scores_cleanliness, review_scores_location, review_scores_value, reviews_per_month
- categorical: property_type, room_type, host_since, host_location, host_response_time, host_response_rate, host_acceptance_rate, city
- text: description, amenities, first_review, last_review
- datetime: not available
- boolean_like: host_is_superhost, has_availability
- id: id
- target: price

## 7. Созданные признаки
- `availability_30_ratio`: Показывает долю краткосрочной доступности относительно годовой, что отражает интенсивность использования объекта.
- `availability_60_ratio`: Дополнительный признак доли доступности на среднем горизонте относительно годовой доступности.
- `review_activity_total`: Сводный индикатор отзывной активности объекта по нескольким связанным метрикам.
- `reviews_ltm_share`: Доля отзывов за последний год от общего числа отзывов, отражает текущую активность.
- `reviews_l30d_share_of_ltm`: Показывает концентрацию недавних отзывов внутри годовой активности.
- `accommodates_per_bed`: Отражает вместимость относительно числа спальных мест, что полезно для оценки плотности размещения.
- `accommodates_per_bedroom`: Показывает вместимость относительно числа спален; может различать компактные и просторные варианты.
- `review_score_mean`: Средняя оценка по связанным review score колонкам агрегирует общий уровень качества объекта.
- `rating_minus_value_score`: Разница между общей оценкой и оценкой ценности может улавливать несоответствие ожиданий цене.

## 13. Бизнес-интерпретация результатов
Модель объясняет около 60% вариации цены, что указывает на наличие выраженных закономерностей в данных, но также показывает, что цена Airbnb зависит и от факторов, которых нет в признаках или которые описаны недостаточно полно.
С практической точки зрения модель может быть полезна для первичной оценки цены, выявления объектов с завышенной или заниженной стоимостью и поддержки аналитики по влиянию характеристик объекта на цену.
Наиболее полезными выглядят признаки, связанные с вместимостью, доступностью, отзывной активностью и качеством объекта по оценкам гостей.
